Write a function that takes in a number and returns the corresponding day of the week.

Ex:
Input: 1
Output: 'Monday'

Input: 5
Output: 'Friday'

Input: 8
Output: undefined

Bonus:
The size of the number doesn't matter.

Ex:
Input: 21
Output: 'Sunday'

Input: 431
Output: 'Thursday'

Input: 1002
Output: 'Monday'
